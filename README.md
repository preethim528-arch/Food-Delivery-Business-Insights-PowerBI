Food Delivery Business Performance — Power BI Dashboard 

Tools: Power BI • Power Query • SQL • DAX  
Duration: 1 week  
Goal: Analyze revenue, delivery speed & customer retention for a food delivery business

## Key Insights
• Only 30% of orders delivered on time → low customer satisfaction  
• Repeat rate = 46% → weak loyalty and high churn risk  
• Slow delivery from high-revenue restaurants → low ratings  
• 53% of revenue comes from regular customers → dependency risk

## Project Workflow
SQL → Power Query → Data Model → DAX → Power BI → Insights & Recommendations
